
1. Follow steps listed in https://cordova.apache.org/docs/en/latest/guide/cli/ ,
   to create a cordova application . Install node.js and install the cordova node module using npm as 
   documented in https://cordova.apache.org/docs/en/latest/guide/cli/ .

2. Extract all the files from the remote github repository into your local machine and place it in a folder 
   named 'Delivery'. 

3. cd into the project folder and follow commands listed in 
   https://cordova.apache.org/docs/en/latest/guide/cli/ to build and run the cordova app in the android
   platform.
